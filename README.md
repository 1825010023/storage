这个项目本来是B站上一位UP用java写的，征求UP的同意后我拿过来用Golang来写后端部分。

项目[演示地址](http://www.pyxgo.cn/ )



### 运行

首先需要在conf文件夹中配置好ini文件

```
go mod tidy
```

```
go run main.go
```

访问localhost:8080
